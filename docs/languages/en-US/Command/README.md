```
mw:home
```

```
[[home]]
```

## /help

Example:
```
/help
```

## /language
::: warning 注意
The modification only takes effect for a single user. Please modify the default language in the configuration file.
:::

1. `/language`

展示可用语言列表

Example:
```
/language
```

2. `/language language`

Change language

Example:
```
/language zh-CN
```

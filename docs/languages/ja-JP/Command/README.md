# コマンド

## 検索Wiki
1. `Wiki名前:検索したいこと`

例:
```
mw:◯◯
```

2. `[[検索したいこと]]`

例:
```
[[◯◯]]
```

## /help
1. `/help`

**ヘルプ**

例:
```
/help
```

## /language
::: warning 注意
変更は1人のユーザーに対してのみ有効。デフォルトの言語を変更するには、構成ファイルで変更してください
:::

1. `/language`

**使える言語リストの表示**
  
例:
```
/language
```

2. `/language 言語`

**言語の変更**

例:
```
/language zh-CN
```
